System Technology

Information Processing Cycle (IPC)

Converting raw data into meaningful information through 5 phases

Definition: The process of converting raw data into meaningful information to support decision-making.

Input

Raw facts, figures, and observations collected

Example: Customer enters product details and payment information

Processing

CPU executes instructions, applies formulas, or analyses data

Example: System verifies payment, updates stock, processes order

Storage

Processed data is saved for future access

Example: Order details saved in database

Output

Processed information presented in usable format

Example: Confirmation message displayed, invoice generated

Communication

Information transmitted between devices, users, or systems

Example: Order confirmation email sent, tracking details shared

Processors (CPU)

The "brain" of the computer that executes instructions and processes data

Definition: The "brain" of the computer; executes instructions, processes data, and controls other hardware.

Clock Speed (GHz)

Measures how fast the CPU processes instructions; higher GHz means faster performance

Cores

Multiple processing units allow running multiple tasks simultaneously (dual-core, quad-core)

Threads

A sequence of instructions the CPU executes; hyper-threading increases efficiency

Fit for Purpose:

Basic Use: Dual-core (Intel i3/AMD Ryzen 3), 2.5-3.0 GHz
Moderate Use: Quad-core (Intel i5/AMD Ryzen 5), 3.0 GHz+
Heavy Use: Octa-core (Intel i7/i9/AMD Ryzen 7/9), 3.5 GHz+

RAM (Memory)

Temporary storage for data and programs currently in use

Definition: Temporary storage for data and programs currently in use, allowing faster access than permanent storage.

Capacity (GB)

More RAM allows more applications to run simultaneously without slowing down

Speed (MHz)

Faster RAM speeds up data access

Type

DDR4 is faster and more energy-efficient than DDR3

Fit for Purpose:

Basic Use: 4-8 GB DDR4
Moderate Use: 8-16 GB DDR4
Heavy Use: 16 GB+ DDR4

SSD vs HDD Storage

Types of non-volatile storage for operating systems and files

Definition: Types of non-volatile storage for operating systems, applications, and files.

SSD (Solid State Drive)

  • Significantly faster read/write speeds
  • More durable (no moving parts)
  • Lower power consumption
  • No fragmentation needed
  • More expensive per GB

HDD (Hard Disk Drive)

  • Lower cost per GB
  • Higher storage capacities
  • Slower read/write speeds
  • Susceptible to physical shock
  • Higher power consumption
  • Requires defragmentation

Network & Internet Technologies

WAN (Wide Area Network)

A network spanning large geographical areas, connecting multiple LANs

Definition: A network that spans a large geographical area, connecting multiple local networks (LANs) across cities, countries, or continents.

Purpose

  • Enable communication across large distances
  • Facilitate data/resource sharing
  • Support remote access
  • Improve business operations
  • Enable cloud computing

How Internet Functions as WAN

  • Connects billions of devices globally
  • Uses wired (fiber optic) and wireless (satellite, cellular) communication
  • Interconnects multiple LANs/MANs
  • Transfers data over long distances using TCP/IP
  • Supports various online services
Example: The Internet is the largest example of a WAN

VPN (Virtual Private Network)

Secure access to company data over the Internet with encrypted connections

Definition: A device that allows users to securely access company data over the Internet. It creates an encrypted connection over a public network.

How it Works

  • Masks actual location
  • Encrypts data transmission
  • Ensures user anonymity
  • Creates secure tunnel through public networks

Advantages

  • Blocks unauthorized access
  • Secures communications
  • Bypasses geo-blocking
  • Prevents data interception
  • Enables secure remote work

Bandwidth vs Broadband

Understanding the difference between connection capacity and connection type

Bandwidth

Definition: The amount of data that can be transmitted over a network in a given time (measured in bps).

  • Affects speed and efficiency of data transfer
  • Measured in Kbps, Mbps, Gbps
  • Determines how much data can flow

Broadband

Definition: High-speed, always-on internet access. It is a type of internet connection.

  • Provides the infrastructure
  • Always-on connection
  • Supports multiple data types
Relationship: Broadband provides the infrastructure, and bandwidth is the capacity of that infrastructure

Reasons for Networks

Why organizations and individuals use networks

Resource Sharing

Sharing hardware (printers, scanners) and software across multiple users

Communication

Fast and efficient communication through email, instant messaging, video calls

Centralized Data Storage

Easier management and backup of data in central locations

Improved Security

Centralized control over data access and security policies

Social Implications

Identity Theft

Using someone else's personal information without permission to commit fraud

Definition: Using someone else's personal information (name, address, financial details) without permission to commit fraud or other crimes.

How it Happens

  • Data breaches from companies
  • Social engineering attacks
  • Phishing emails and websites
  • Unauthorized sharing of personal data
  • Skimming devices on ATMs

Impact

  • Financial fraud and loss
  • Damage to credit reputation
  • Legal complications
  • Emotional stress and anxiety
  • Time-consuming recovery process

Ergonomics

Designing environments to fit people, focusing on comfort and preventing injury

Definition: The science of designing and arranging products and environments to fit the people who use them, focusing on comfort and preventing injury.

Computer Setup

  • Monitor at eye level
  • Proper chair height and support
  • Keyboard at comfortable height
  • Mouse placement within easy reach
  • Adequate lighting to reduce eye strain

Health Benefits

  • Reduces physical strain and fatigue
  • Improves posture and spinal health
  • Increases comfort and productivity
  • Prevents repetitive strain injuries
  • Reduces eye strain and headaches
Best Practice: Take regular breaks, sit up straight, stretch regularly, and look away from screen every 20 minutes

Malware Types

Malicious software designed to harm or exploit devices and networks

Definition: Malicious software designed to harm or exploit a device, network, or server.

Virus

Replicates itself, spreads to other files, and causes damage or steals data

Trojan

Disguised as legitimate software but allows unauthorized access to systems

Worms

Self-replicating malware that spreads across networks without user interaction

Ransomware

Encrypts user data and demands payment for decryption key

Spyware

Secretly gathers information about users and their activities

Protection Methods:

  • Antivirus/anti-malware software
  • Firewalls and network security
  • Email filters and spam protection
  • Regular software updates
  • Safe browsing practices

Cryptocurrency as Payment

Digital currencies using cryptography, operating independently of central banks

Definition: Digital or virtual currencies that use cryptography for security, operating independently of central banks (e.g., Bitcoin, Ethereum).

Advantages

  • Lower transaction fees
  • Fast international transfers
  • Global accessibility
  • No intermediaries needed
  • Investment opportunities

Disadvantages

  • High price volatility
  • Limited merchant acceptance
  • Technical complexity
  • Regulatory uncertainty
  • Environmental concerns
Impact: Financial inclusion for unbanked populations and alternative investment opportunities

Solution Development

HTML Features

Hypertext Markup Language used to structure content on the web

Definition: Hypertext Markup Language, used to structure content on the web.

Lists

Ordered (<ol>) and unordered (<ul>) lists to organize information

Tables

<table> element to display data in rows and columns

Excel Features

Spreadsheet application for data analysis, calculations, and visualization

Functions and Formulas

  • SUM: Adds up values in a range
  • AVERAGE: Calculates mean of values
  • COUNT: Counts cells with numbers
  • IF: Conditional logic statements
  • VLOOKUP/HLOOKUP: Search and retrieve data

Key Features

  • Fill Handle: Copy formulas to adjacent cells
  • Relative vs Absolute: Cell reference behavior when copying
  • Charts: Visual data representation
  • Pivot Tables: Data summarization and analysis

Access Features

Database management system for storing and organizing data

Input Masks

String of characters that dictates format for data entry into a field

Examples: Phone numbers (000) 000-0000, Dates MM/DD/YYYY

Query Criteria

Conditions used in queries to filter and retrieve specific data from tables

Table Properties

Characteristics of fields in database table (data type, field size, validation rules)

Benefits of Input Masks:

  • Data consistency and standardization
  • Reduced data entry errors
  • User-friendly data entry
  • Professional appearance

Working together on projects using shared tools and platforms

Collaboration Tools

Feature that records edits (insertions, deletions, formatting changes) made to a document, allowing multiple users to review and accept/reject changes.

Information Management

Evaluation of Sources

Critically assessing the reliability and credibility of information sources

Definition: Critically assessing the reliability and credibility of information sources.

Accuracy

Is the information factual and free from errors?

Authority/Credibility

Who is the author? Are they an expert? Is the source reputable?

Objectivity

Is the information biased or impartial?

Currency

Is the information up-to-date and relevant?

Coverage

Is the information comprehensive and complete?

Referencing Sources

Acknowledging original creators to avoid plagiarism and give proper credit

Definition: Acknowledging the original creators of information used in your work to avoid plagiarism.

Information to Include

  • Author/creator/photographer/artist name
  • Title of the work or image
  • Website/hyperlink to original source
  • Date of creation/publication
  • Publisher or organization

Why It's Important

  • Avoid plagiarism and academic dishonesty
  • Give proper credit to original creators
  • Enable readers to verify information
  • Maintain academic and professional integrity
  • Respect intellectual property rights

Types of Graphs
